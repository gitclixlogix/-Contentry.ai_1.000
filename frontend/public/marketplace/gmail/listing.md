## Permissions Justification

### gmail.compose
Required to access the email content (subject and body) during composition for real-time analysis.

### gmail.addons.execute
Required to run the add-on within Gmail interface.

### gmail.addons.current.message.readonly
Required to read the current draft message for analysis.

### script.external_request
Required to communicate with Contentry.ai API servers for content analysis.
